# 🛒 E-Commerce Cart 

A dynamic shopping cart web application built using HTML, CSS, and JavaScript with cart and wishlist functionality.

## 🚀 Features

- Add products to cart
- Increase / decrease quantity
- Remove items from cart
- Clear entire cart
- Wishlist functionality
- Move items from wishlist to cart
- Total price calculation
- Data persistence using localStorage

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

## 🧠 Key Concepts Used

- DOM Manipulation
- Event Listeners
- Arrays & Objects
- localStorage
- State Management
- Dynamic Rendering

## 💡 How to Use

1. Click "Add to Cart" to add items
2. Use + / - buttons to change quantity
3. Remove items if needed
4. Add items to wishlist and move to cart
